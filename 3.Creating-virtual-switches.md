## ** Create Virtual Switches** 

1. Open **Hyper-V Manager** â†’ in the right-pane, click on **Virtual Switch Manager**. A wizard should pop-up , under Virtual Switches, clic on 'New virtual network switch'. to the right select 'Internal' and then 'Create Virtual Switch'
2. Create two virtual switches:  
   - **Internal Network** (e.g., `Lab-Internal`)  
   - **External Network** (e.g., `Lab-External`).  
     NB:The external is already created for you. You will only need to change the name to suit your preference.
